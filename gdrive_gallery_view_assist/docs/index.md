# Privacy Policy

Last updated: 2026-04-20

## Overview
This project provides a self-hosted Home Assistant add-on that retrieves images from a Google Drive folder to serve as a local image endpoint. The add-on runs on your own Home Assistant system and is not operated as a hosted service.

## Data We Access
When configured by the user, the add-on accesses:
- Google Drive image files in the folder specified by the user
- File metadata necessary to list and download those images (file IDs, names, and mime types)

## Data We Do Not Collect
We do not collect, store, sell, or share personal data.
We do not use analytics, tracking pixels, or third-party telemetry.

## Data Storage
- The add-on may cache images in memory to improve performance.
- Configuration is stored locally on the user's Home Assistant system.
- The add-on does not transmit data to any third parties except Google APIs required to access the configured Google Drive folder.

## Data Sharing
Data is only exchanged between the user's Home Assistant instance and Google APIs. No data is sent to the project author or any other external services.

## User Control
You can revoke access at any time from your Google Account security settings or by removing the add-on configuration.

## Changes
This policy may be updated to reflect changes to the add-on. Updates will be published in this repository.
